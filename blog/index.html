Muhammed Zain Shared his wild fire prediction project, which uses LoRa radio communication methods to transfer the information and LoCom, That is build for remote communities for emergency communication. , and Travis shared a project called One Planet Education where the LoCom project can be helpful and shared a similer project called Off-grid LoRa Communicator that similar to LoCom.

Travis then shared about the open thread project and maker youtube channel, and that led the discussion to ESP-NOW Protocol
